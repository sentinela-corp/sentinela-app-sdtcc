# Projeto Sentinela - Global Solution

Plataforma de tecnologia ambiental e monitoramento inteligente aplicada à Indústria Espacial.

## 🚀 Sobre o Projeto
A **Sentinela** atua como uma "sentinela digital", utilizando inteligência artificial para processar dados de satélites e informações climáticas em tempo real. Nosso objetivo é identificar regiões críticas, prever situações de risco e automatizar alertas preventivos contra desastres ambientais, como queimadas.

Este projeto está alinhado ao **ODS 13 (Ação Climática)**, demonstrando como a infraestrutura espacial e de nuvem pode mitigar impactos ambientais na Terra.

## 🛠️ Tecnologias e Arquitetura (DevSecOps)
Este projeto foi desenvolvido utilizando as melhores práticas de **Secure DevOps**:
* **Cloud Platform:** Microsoft Azure (App Service, Key Vault, Application Insights).
* **CI/CD:** Pipeline automatizado via GitHub Actions.
* **Segurança:** Gestão de segredos com Azure Key Vault e GitHub Secrets.
* **Monitoramento:** Application Insights e alertas configurados para falhas críticas.
